## What I Have Built

### FlowDesk — AI Productivity System

> A full-stack AI system that monitors cognitive presence using computer vision, classifies browser activity using a self-trained transformer model, enforces focus through a Chrome extension, and responds to natural language voice commands through an LLM-powered assistant.

**What makes it real engineering:**

- Collected and labelled a dataset of **2474 browser activity samples** from scratch
- Fine-tuned **DistilBERT** (66M parameters) on that dataset achieving **99.46% accuracy**
- Deployed the model as a **local Flask REST API** serving live predictions with confidence scores
- Built a **Chrome Extension (Manifest V3)** that monitors tabs every second and can open or close them programmatically
- Integrated **TensorFlow.js + BlazeFace** for real-time face detection running entirely in the browser — detecting active, away, and absent states
- Built an always-listening **voice assistant powered by Groq LLaMA 3.3-70b** that understands any natural language command, speaks automatic alerts, and closes distracting tabs with a countdown
- Everything runs at **$0.00** — no paid APIs, no cloud compute, no subscriptions

```
Custom Dataset → DistilBERT Fine-tuning → Flask API → React Dashboard
     2474 rows       99.46% accuracy       localhost      live confidence scores
```

**Stack:** React · TensorFlow.js · DistilBERT · PyTorch · HuggingFace · Flask · Chrome MV3 · Firebase · Groq API · Browser Speech APIs
